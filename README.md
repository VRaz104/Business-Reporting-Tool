# Business-Reporting-Tool

This Python script analyzes sales data from a CSV file, calculates key business metrics, plots daily revenue trends, and saves the outputs with timestamped filenames. It includes robust error handling, logging, and a configuration file for easy customization.

## Features

- Calculates **Total Revenue, Total Cost, Total Profit, Profit Margin**
- Generates a **daily revenue trend plot**
- Saves **summary CSV** and **plot image** with timestamped filenames
- Configurable through a **JSON config file**
- Logging of all steps and errors for easy debugging

## Requirements

- Python 3.8+
- Libraries: `pandas`, `matplotlib`
